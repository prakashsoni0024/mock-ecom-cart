# Mock E-Com Cart

This is a simple full-stack **e-commerce cart application** where users can browse products, add or remove items from the cart, update quantities, and view the total bill. The project demonstrates basic **frontend-backend communication** and standard e-commerce cart functionality.

---

## Features

- View a list of products
- Add items to the cart
- Remove items from the cart
- Update the quantity of selected products
- Display total amount of the cart
- Mock checkout (no actual payment integration)
- Backend data served through REST APIs

---

## Tech Stack

| Layer      | Technology Used           |
|------------|---------------------------|
| Frontend   | React.js, TailwindCSS     |
| Backend    | Node.js, Express.js       |
| Database   | MongoDB                   |
